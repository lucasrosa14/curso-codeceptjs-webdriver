# Automação de testes com Codeceptjs + Web Driver

## Projeto criado na Tilha de QA do QAzando

### Instalar dependências

```
git clone https://github.com/lucasrosa14/curso-codeceptjs-webdriver.git
```
```
npm install
```

### Rodar testes

```
npx codeceptjs run --steps
```
